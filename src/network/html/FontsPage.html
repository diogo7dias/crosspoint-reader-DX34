Install a New Font

Drop a TTF or OTF for each variant you want. Only Regular is required — Bold, Italic and Bold Italic are synthesised from Regular when you leave them empty. Choose one or more sizes (25–40 pt); the browser rasterises every glyph in the font at each size, compresses the output and streams it to the device. The device reads glyphs from the SD card on demand, so even big sizes don't pin RAM.
Letters, digits, - and _; must start with a letter or digit; max 32 chars.
