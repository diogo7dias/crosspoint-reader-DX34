# Device and System Controls

## Side buttons

| Setting | Options |
|---------|---------|
| Side button layout | Previous/Next or Next/Previous |
| Long-press chapter skip | On / Off — long-press a side button to skip chapters |

## Front buttons

Front button roles (Back, Confirm, Left, Right) can be remapped to any physical front button via an interactive flow: the firmware prompts for each role and you press the button you want assigned.

Presets are also available for common swaps without entering the remap flow.

## Power button

| Setting | Options |
|---------|---------|
| Short power button | Ignore, Sleep, Page Turn, Refresh Screen |

A short press performs the chosen action; long-press retains its hardware function.

## Display and power

| Setting | Options |
|---------|---------|
| Auto-sleep timeout | 1, 5, 10, 15, 30 minutes |
| Refresh frequency | Full refresh every 1, 5, 10, 15, 30 pages |
| Sunlight fading compensation | On / Off |
| Custom boot image | Place `/boot.bmp` on the SD card |
| Show hidden files | Show files/folders starting with `.` in the browser |
| Random book on boot | Open a random book from recents instead of the last book |

## Orientation

Four modes, set globally or toggled from the in-book menu:

- Portrait (default)
- Landscape CW
- Inverted
- Landscape CCW
