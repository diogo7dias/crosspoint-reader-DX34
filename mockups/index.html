CROSSPOINT ─ WEB UI // MOCKUPS

v0.MOCK · 2026.04

Five alternative directions for the embedded web interface. All share a dark background, drifting pixel squares, and 1px dashed borders — each takes the brief in a different visual direction.

01 / 05
CRT TERMINAL
Green phosphor. Scanlines. Blinking cursor. A literal 80s serial console dressed up as a dashboard.
02 / 05
JP BBS ─ CONDENSED
Dense vertical bands, katakana ornaments, tight monospace, magenta + cyan spot colour. A 2ch-flavoured control panel.
03 / 05
KR PORTAL ─ MULTIWINDOW
Naver-era busy portal. Many small panes, ticker bars, badges, mixed Hangul + English labels.
04 / 05
VAPOR / PIXEL GRID
Magenta-cyan gradient, large drifting pixel blocks, hiragana/kanji accents, modern type mixed with bitmap.
05 / 05
BRUTALIST / 8-BIT
Flat, chunky, oversized labels. Huge numerals, thick dashed dividers, pixel-art status glyphs.
